## NLU Assignment 2

This repository contains solutions for the following problems:

### PROBLEM 1: LEARNING WORD EMBEDDINGS FROM IIT JODHPUR DATA  
Implemented word embedding models using curated text data from IIT Jodhpur sources. The notebook includes preprocessing, training, and evaluation.

### PROBLEM 2: CHARACTER-LEVEL NAME GENERATION USING RNN VARIANTS  
Implemented character-level name generation using different sequence models (RNN variants). The notebook includes training and generation of names.
